沒想出來的題目們

第一題

\(\sum_{k=1}^{\infty}(-1)^k \frac{1}{4k^2-1} = ?\)

思路

這題應該會先觀察到 \(4k^2-1 = (2k+1)(2k-1)\)

接下來要想到 \(\frac{1}{(a-1)(a+1)} = \frac{1}{2} (\frac{1}{a-1} - \frac{1}{a+1})\)

化簡之後就會得到長得像 \(1 - \frac{1}{3} + \frac{1}{5} - \frac{1}{7} ...\) 的東西

做到這邊，我就不會了

解答

其實 \(1 - \frac{1}{3} + \frac{1}{5} - \frac{1}{7} ...\) 就等於 \(\frac{\pi}{4}\)

雖然這大概不太可能想到，但我還是紀錄一下天才會怎麼想

首先你要發現 \(x - \frac{x^3}{3} + \frac{x^5}{5}...\) 是 \(1 - x^2 + x^4...\) 的積分

然後 \(1 - x^2 + x^4... = \frac{1}{1+x^2}\)（這好像不太嚴謹，因為這只在 \(|x^2|<1\) 的時候成立）

如果你是天才大概會知道 \(\frac{1}{1+x^2}\) 是 \(arctan\) 的微分

所以實際上 \(x - \frac{x^3}{3} + \frac{x^5}{5}...\) 就是 \(arctan\)

然後把 \(1\) 帶進去就會得到 \(\frac{\pi}{4}\) 了。

心得

最近覺得算數學真的是大型通靈現場，感覺只要想錯一步就找不出答案了
現在想題目都會提醒自己要先找到題目奇怪的地方、卡住的時候要檢查自己有什麼資訊 但還是很容易想不到 QQ
說不定只能靠經驗來彌補自己的笨。

第二題

\(\bigtriangleup ABC\), 已知 \((a − c)(\sin A + \sin C) = (a − b)\sin B\)
1. 求 \(\angle C =?\)
2. 求 \(\sin A \sin B\) 的最大值

思路

這題我真的沒想法，用正弦定理亂做一通還是找不到路

解答

哈哈想不到吧 ～ 這題就是正弦定理

基本上就是把題目給的先變成 \((a-c)(a+c) = (a-b)b\)

展開以後 \(a^2 - c^2 = ab - b^2\)

這時候有個重要的觀察是，這長得很像餘弦定理

然後就做一做就可以算出 \(\angle C\) 是多少了。

至於第二小題，要先想到 \(\sin A \sin B = \frac{1}{2} (\cos (A - B) - \cos (A + B))\)

因為是三角形，所以 \(\cos (A+B) = -\cos C\)，然後就可以算出答案了。

心得

這題沒做出來沒主要是因為沒想到 \(\sin A\) 正比於 \(a\)，
但感覺餘弦的部分我大概也想不到，可能是對三角函數太不熟了， 所以還是只能多練

第三題

過 \((1, 3)\) 的動直線與圓 \(x^2+y^2=4\) 的兩交點的兩切線交於 \(Q\) 點
求動點 \(Q\) 軌跡方程式

思路

原本是想用切線和圓心會形成直角三角形的特性來做，但好像沒辦法。

解答

這題的做法是先假設那兩個交點是 \((a_1, b_1)\) , \((a_2, b_2)\)

算出切線後，用克拉瑪找到兩切線的交點（或是直接用算的）

然後最後再利用三點共線的性質，算出軌跡方程式。

心得

這題的精髓大概是假設座標和從三點共線的式子裡分離出切線的交點，
有時候不要直接用性質來假設好像會比較好算，
但感覺還是得通靈 ...

小結
